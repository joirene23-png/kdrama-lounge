K-Drama Lounge is a fan-powered destination for K-drama recaps, clips, reviews, recommendations, and the latest updates from the world of Korean dramas.

## 🌸 About the Project

This website is built to share and celebrate Korean dramas with fellow fans. It features:
- Episode recaps and summaries
- Drama reviews and ratings
- Clip highlights and commentary
- Recommendations for what to watch next
- Latest K-drama news and updates
